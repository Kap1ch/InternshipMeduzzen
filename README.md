# InternshipMeduzzen

## Start project

#### Clone repository

`git clone https://github.com/username/repository.git`

#### Create new environment

`python -m venv venv `

#### activate environment

On Windows

`venv\Scripts\activate`

MacOS & Linux

`source venv/bin/activate`

#### Install packages

`pip install -r requirements.txt`
